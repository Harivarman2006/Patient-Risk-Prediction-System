Model Technical Details

The risk engine uses a Logistic Regression model with L2 regularization trained on 5,000 synthetic patient records. The synthetic records statistically mimic patterns observed in the clinical UCI Diabetic Readmission dataset.

To generate predictions in the web browser, patient features are standardized and computed using the logistic sigmoid function:

$$P(\text{Readmission}) = \sigma(\mathbf{w}^T \mathbf{x} + b) = \frac{1}{1 + e^{-(\mathbf{w}^T \mathbf{x} + b)}}$$

Where:

  • \(\mathbf{x}\) is the feature vector (preprocessed matching the training pipeline)
  • \(\mathbf{w}\) is the model weights (coefficients)
  • \(b\) is the intercept (\(-0.7837\))
